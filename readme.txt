App每日推送每日最新推荐挂件，iPhone/iPad 限时免费精选

== Description ==

在你的博客显示最新的精选 iOS App。

本插件数据由中国领先的 iPhone/iPad App推荐网站 http://appdp.com/ 提供。

== Installation ==

1. 解压后把 `appdp-list` 目录上传到 /wp-content/plugins
2. 在后台激活 `Appdp List` 插件
3. 到 `主题` -> `小工具` 页面，把 `iOS应用挂件` 拖到左边的侧边栏里
